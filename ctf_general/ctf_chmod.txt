Purpose:
Modify read (r), write (w), and execute (x) permissions on files and directories.

Basic Permission Types:
Permission	Meaning
r	Read permission
w	Write permission
x	Execute permission (or enter directory)

User Categories:
Symbol	Meaning
u	User (file owner)
g	Group
o	Others (everyone else)
a	All (user, group, others)


 ***The chmod +x command in Linux adds the execute (x) permission to a file. This permission allows the file to be executed as a program or script.***
